## 🕹️ ゲームの遊び方と操作方法

キーボードを使ってブロックを操作し、ラインを消してハイスコアを目指しましょう！

| 操作 | キー | 役割 |
| :--- | :--- | :--- |
| **左移動** | **←** (左矢印キー) | ブロックを左に動かします。 |
| **右移動** | **→** (右矢印キー) | ブロックを右に動かします。 |
| **加速落下** | **↓** (下矢印キー) | ブロックを素早く下に落とします。 |
| **回転** | **↑** (上矢印キー) | ブロックを回転させます。 |

## ✨ ゲームの特徴

このテトリスには、難易度を上げるためのシステムと、サプライズ要素が組み込まれています。

### 📊 スコアとレベルシステム

* **ライン消去**：ブロックを消すとスコアが加算されます。
* **レベルアップ**：スコアが一定値に達すると**レベルが上がり**、ブロックの**落下速度が速く**なります。

### 💎 レアブロック（サプライズ）

* **出現**：時々、通常のブロックとは違う**鮮やかなピンク色**のブロックがランダムで出現します。
* **固定時**：このピンク色のブロックは、盤面に固定されると**白色**に変化します。
* **ボーナス**：この**白色のレアブロック**が含まれるラインを消すと、通常のスコアに加えて**500ポイントのボーナス**が加算されます。レアブロックを狙ってラインを消しましょう！
